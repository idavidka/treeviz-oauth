# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Initial release of @treeviz/oauth package
- TreeVizOAuth client class for OAuth 2.0 authentication
- Popup-based authentication flow
- Framework-agnostic design (no Firebase dependency)
- TypeScript support with full type definitions
- Comprehensive documentation and examples
- Configurable OAuth parameters (scopes, popup size, etc.)
- Automatic token handling and error management
- Support for development and production environments

### Features
- OAuth 2.0 authentication with TreeViz
- Custom token generation for Firebase integration
- Secure popup-based flow
- User profile data retrieval (email, displayName, photoURL)
- Configurable scopes (email, profile)
- Error handling with descriptive messages
- Automatic popup cleanup
